## 说明

* 软件不定期同步大神库更新，适合一键下载到package目录下，用于openwrt编译

* lede/package$下运行

```bash
 git clone https://github.com/kenzok8/openwrt-packages.git
```

- openwrt 固件编译自定义主题与软件
- luci-app-openclash           
- clash图形简易版
- luci-theme-atmaterial        
- 抹茶绿黑.红黑, 暗黄黑三套主题
- luci-theme-argon-dark
-  argon高亮主题 
- luci-app-adguardhome         
- DNS防污染去广告
- luci-app-passwall            
- Lienol大神passwall
- luci-app-dockerman           
- 图形化Docker界面
